## 📱 **FORMAT EXAMPLES**

### **Kanji with Furigana:**
```
助詞(じょし)練習(れんしゅう) [Particle Practice]
- 助詞 = particle
- 練習 = practice
```

### **Katakana Only:**
```
スライドパズル [Slide Puzzle]
- スライド = slide (loanword)
- パズル = puzzle (loanword)
```

### **Mixed Format:**
```
文字(もじ)を書(か)こう [Character Trace]
- 文字 = character/letter
- 書こう = let's write
```

### **Compound Words:**
```
電話(でんわ)応対(おうたい) [Phone Calls]
- 電話 = telephone
- 応対 = reception/response
```
