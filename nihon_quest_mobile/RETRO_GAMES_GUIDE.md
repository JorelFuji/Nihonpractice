## 🔄 **HOW TO UPDATE**

### **To Add More Games:**

1. Open `retro_games_screen.dart`
2. Add game card using `_buildGameCard()`:

```dart
_buildGameCard(
  context,
  icon: '🎮',
  title: 'Game Name (Year)',
  subtitle: 'Japanese Name - Description',
  era: 'Era Label',
  color: Colors.blue,
  onTap: () => _showComingSoon(context, 'Game Name'),
),
```

3. Rebuild and deploy
