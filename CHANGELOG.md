## How to Update This File

When making changes:

1. Add entries under `[Unreleased]` section
2. Use categories: Added, Changed, Deprecated, Removed, Fixed, Security
3. Write in present tense ("Add feature" not "Added feature")
4. Link to issues/PRs when relevant
5. On release, move `[Unreleased]` to new version section

Example:
```markdown
### Added
- New Pokemon game (#123)
- Voice recognition for pronunciation (#124)

### Fixed
- Mobile layout issue on iOS (#125)
```
