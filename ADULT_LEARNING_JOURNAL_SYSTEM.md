## 📝 **Fill in the Blanks**

### **How It Works:**

**1. See sentence with blank:**
```
私___学生です。
わたし___がくせいです
```

**2. Choose from 4 options:**
```
は  が  を  に
```

**3. Get instant feedback:**
```
✓ Correct!
は marks the topic of the sentence.
"As for me, (I am) a student."
```

**4. Hear full sentence** 🔊

**5. Score +10 points!**

---

### **Exercise Examples:**

| Sentence | Correct Answer | Level |
|----------|----------------|-------|
| 私___学生です | は | N5 |
| 寿司___食べます | を | N5 |
| 図書館___勉強します | で | N5 |
| レックス___パンツ___忘れないでください | の...を | N4 |

## 🏗️ **Sentence Builder**

### **How It Works:**

**1. See English sentence:**
```
English: I eat sushi
💡 Hint: Remember: Japanese is SOV
```

**2. Available words:**
```
[私は] [寿司を] [食べます] [。]
```

**3. Tap to build:**
```
Your sentence: 私は 寿司を 食べます 。
```

**4. Check answer:**
```
✓ Perfect sentence! +15 points!
```

## 💬 **Conversation Practice**

### **How It Works:**

**1. See context:**
```
Context: At a restaurant, ordering food
```

**2. See prompt:**
```
Server asks: ご注文は？(Your order?)
🔊 [Hear pronunciation]
```

**3. Type your response:**
```
Your response: [text box]
すみません、寿司をください。
```

**4. Get feedback:**
```
✓ Good response! +20 points!

Sample answer:
すみません、寿司をください。

Explanation:
ください is the polite way to request something.
すみません gets attention first.

Other good responses:
• 寿司をお願いします。
• これをください。
```

### **Exercise Data Structures:**

```typescript
interface FillBlankExercise {
  id: string
  sentence: string           // 私___学生です
  kana: string              // わたし___がくせいです
  blank: string             // は
  options: string[]         // ['は', 'が', 'を', 'に']
  correctAnswer: string     // 'は'
  explanation: string       // Grammar explanation
  level: 'N5' | 'N4' | 'N3'
}

interface SentenceBuilderExercise {
  id: string
  english: string           // "I eat sushi"
  words: string[]          // ['私は', '寿司を', '食べます']
  correctOrder: string[]   // Same as words (in correct order)
  hint: string            // "Remember: SOV"
  level: 'N5' | 'N4' | 'N3'
}

interface ConversationExercise {
  id: string
  context: string          // "At a restaurant"
  prompt: string          // "Server: ご注文は？"
  correctResponse: string // "すみません、寿司をください。"
  alternatives: string[]  // Other acceptable answers
  explanation: string     // Why this works
  level: 'N5' | 'N4' | 'N3'
}
```
