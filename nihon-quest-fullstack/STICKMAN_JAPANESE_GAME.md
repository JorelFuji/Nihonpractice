# 🏃 Stickman Japanese Adventure - Action Learning Game

## 🎯 **Game Concept**

**"Stickman Sensei" (スティックマン先生)** - A fast-paced action platformer where kids learn Japanese through movement, combat, and exploration!

Think: **Stickman games + Duolingo + Ninja action**

---

## 🎮 **Core Gameplay**

### **Genre: Action Platformer**
- Side-scrolling levels
- Run, jump, climb, fight
- Collect Japanese characters
- Defeat enemies with words
- Boss battles with grammar challenges

### **Learning Integration**
- **Movement**: Jump on hiragana platforms
- **Combat**: Attack enemies with Japanese words
- **Collection**: Gather kanji power-ups
- **Puzzles**: Solve grammar to unlock doors
- **Boss Fights**: Answer questions to damage boss

## 🎯 **Combat System (Learn by Fighting)**

### **Word-Based Combat**

```
Attack with Japanese words!

Basic Attack: Punch
- Tap enemy
- Shows random hiragana
- Enemy takes damage

Special Attacks: Words
Collect word cards to unlock:

あか (aka / red) → Fire punch 🔥
あお (ao / blue) → Ice punch ❄️
みず (mizu / water) → Water blast 💧
かぜ (kaze / wind) → Wind slash 💨

Combo Attacks: Sentences
Combine words for powerful moves:

ねこ (cat) + いぬ (dog) = Animal combo! 🐱🐕
あか (red) + あお (blue) = Purple power! 🟣
```

### **Enemy Types**

```
1. Confused Character (?)
   - Weak enemy
   - Defeat by showing correct character
   - Drops: 1 hiragana

2. Mispronounced Word (むずかしい)
   - Medium enemy
   - Defeat by pronouncing correctly
   - Drops: Word card

3. Grammar Goblin (文法)
   - Strong enemy
   - Defeat by completing sentence
   - Drops: Grammar power-up

4. Kanji Monster (漢字)
   - Boss enemy
   - Defeat by reading kanji
   - Drops: Rare kanji scroll
```

### **Boss Battles**

```
Epic boss fights with Q&A!

Example: Hiragana Dragon

Phase 1: Attack phase
- Dodge dragon attacks
- Hit weak spots
- Build up special meter

Phase 2: Question phase
Dragon asks: "What is this character?"
Shows: あ

Options:
[a] [i] [u] [e]

Correct answer = Big damage!
Wrong answer = Dragon attacks!

Defeat boss = Unlock next world!
```

## 🎯 **Learning Mechanics**

### **1. Character Collection**

```
Collect all Japanese characters!

Hiragana: 46 characters
- Basic: あいうえお (5)
- K-row: かきくけこ (5)
- S-row: さしすせそ (5)
- ... (continue for all rows)

Katakana: 46 characters
- Basic: アイウエオ (5)
- K-row: カキクケコ (5)
- ... (continue for all rows)

Kanji: 80 basic characters
- Numbers: 一二三四五
- Nature: 山川木火水
- Body: 目耳口手足
- ... (JLPT N5 kanji)

Display in collection book:
✓ = Learned
? = Not yet found
🌟 = Mastered (used 10+ times)
```

### **2. Word Building**

```
Combine characters to make words!

Found: ね + こ
Combine: ねこ (neko / cat) 🐱

Found: い + ぬ  
Combine: いぬ (inu / dog) 🐕

Found: さ + か + な
Combine: さかな (sakana / fish) 🐟

Each new word = Special attack unlocked!
```

### **3. Sentence Challenges**

```
Build sentences to solve puzzles:

Puzzle: Cross the bridge
Need: Complete sentence

Given words:
[これ] [は] [ねこ] [です]

Correct order:
これ は ねこ です
(kore wa neko desu)
(This is a cat)

Bridge appears! ✓
```

### **4. Speed Drills**

```
Quick-time events!

Flash cards appear:
あ → Type "a" → Next!
い → Type "i" → Next!
う → Type "u" → Next!

Faster = More points!
Combo meter builds!

10 correct in a row = FEVER MODE! 🔥
- Double points
- Invincibility
- Speed boost
```

### **5. Memory Games**

```
Simon Says - Japanese style!

NPC shows sequence:
あ → い → う

Player repeats:
Tap あ → Tap い → Tap う

Correct = Next level (longer sequence)
Wrong = Try again

Level 10 = Master! 🏆
```
