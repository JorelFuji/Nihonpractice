# 🎮 NihonCraft Kids - Multilingual Japanese Learning Game (Ages 4-10)

## 🎯 **Core Design Principles**

### **Age-Appropriate Learning (4-10 years old)**
- **Ages 4-6**: Hiragana, basic words, colors, numbers
- **Ages 7-8**: Katakana, simple sentences, daily vocabulary
- **Ages 9-10**: Kanji introduction, grammar basics, reading

### **Multilingual Display (Always Visible)**
Every element shows **ALL FOUR** formats simultaneously:
```
┌─────────────────────────────┐
│   木                        │  ← Kanji (if applicable)
│   き                        │  ← Hiragana
│   キ                        │  ← Katakana (if applicable)
│   ki                        │  ← Hepburn Romaji
│   Tree / Wood               │  ← English
│                             │
│   [🔊 Listen]               │  ← Audio pronunciation
└─────────────────────────────┘
```

### **Co-op Parent-Child Play**
- **1 Player Mode**: Kid plays solo with AI helper
- **2 Player Mode**: Parent + child work together
- **Adult Mode**: Parents can practice N2-N1 while helping kids

## 🎮 **Game Modes**

### **Mode 1: Adventure Mode (Story-Driven)**
**Single Player or Co-op**

**Story**: Help Neko-chan (猫ちゃん) rebuild the Japanese village!

**Gameplay Loop:**
1. **Explore** colorful Japanese world
2. **Meet** friendly animal NPCs
3. **Learn** new words/characters
4. **Complete** simple tasks
5. **Build** village buildings
6. **Unlock** new areas

**Example Quest (Age 4-6):**
```
Quest: "Help Usagi-san find carrots"
うさぎさん (Usagi-san) / Rabbit-san

Task 1: Learn colors
- あか (aka) / red
- あお (ao) / blue  
- きいろ (kiiro) / yellow

Task 2: Find 3 red carrots (にんじん / ninjin / carrot)
- Tap red carrots in the garden
- Each tap: "あか！" (aka! / Red!)

Task 3: Give to Usagi-san
- Usagi-san: "ありがとう！" (arigatou! / Thank you!)

Reward: Carrot seeds to plant in your garden
```

### **Mode 2: Mini-Games (Quick Learning)**
**Single Player or Competitive 2-Player**

#### **A. Hiragana Matching Game**
```
┌─────────────────────────────────────┐
│  Match the character to the sound!  │
│                                     │
│  [あ]  [い]  [う]  [え]  [お]      │
│                                     │
│  🔊 "a"   🔊 "i"   🔊 "u"          │
│                                     │
│  Drag hiragana to matching sound    │
│  Parent helps: "That's 'a' - ah!"   │
└─────────────────────────────────────┘
```

**2-Player Mode:**
- Take turns matching
- First to 10 correct wins
- Parent can play with harder characters (katakana/kanji)

#### **B. Word Building Blocks**
```
Build Japanese words with blocks!

Example: "Cat"
┌───┐ ┌───┐
│ ね │ │ こ │
│ ne │ │ ko │
└───┘ └───┘

= ねこ (neko) = Cat 🐱

Child places blocks
Parent reads: "ne-ko, cat!"
```

#### **C. Kana Racing Game**
```
Race through hiragana/katakana course!

Player 1 (Child): Hiragana track
Player 2 (Parent): Katakana track

Obstacles show characters:
- Say it correctly = speed boost
- Wrong answer = slow down
- First to finish wins!
```

#### **D. Memory Match**
```
Classic memory game with Japanese twist

Cards show:
Front: Image (🍎)
Back: Japanese word
- りんご (ringo)
- リンゴ (ringo)  
- ringo
- Apple

Match pairs together
2-player: Take turns, most pairs wins
```

### **Mode 3: Creative Building Mode**
**Co-op Recommended**

```
Build Japanese village together!

Buildings to create:
- いえ (ie / house) 🏠
- がっこう (gakkou / school) 🏫
- こうえん (kouen / park) 🌳
- みせ (mise / shop) 🏪

Each building teaches vocabulary:
House: Family words (おかあさん、おとうさん)
School: School words (せんせい、ほん)
Park: Nature words (き、はな、とり)
```

**Parent-Child Interaction:**
- Child: "I want to build a house!"
- Parent: "Great! In Japanese, house is いえ (ie)"
- Child places blocks labeled いえ
- Both: "いえ！" (ie! / House!)

## 📚 **Curriculum Structure (N5 → N1)**

### **Level 1: Preschool (Ages 4-6) - Foundation**

#### **Hiragana (46 characters)**
```
Taught in groups of 5, with songs and animations

Group 1: あ (a), い (i), う (u), え (e), お (o)
- Each character is an animal friend
- あ = Ahiru (duck) 🦆
- い = Inu (dog) 🐕
- う = Usagi (rabbit) 🐰

Learning method:
1. Watch character animation
2. Trace with finger (touch screen)
3. Hear pronunciation
4. Match to picture
5. Play mini-game
```

#### **Basic Vocabulary (100 words)**
```
Categories:
- Colors: あか、あお、きいろ、みどり
- Numbers: いち、に、さん、し、ご
- Family: おかあさん、おとうさん、おにいさん
- Animals: ねこ、いぬ、とり、さかな
- Food: りんご、バナナ、おにぎり
- Body: あたま、て、あし、め、くち
```

**Display Format:**
```
┌─────────────────────────────┐
│   🐱                        │
│   ねこ                      │  ← Hiragana (main focus)
│   neko                      │  ← Romaji
│   Cat                       │  ← English
│   [🔊]                      │  ← Audio
└─────────────────────────────┘
```

### **Level 2: Early Elementary (Ages 7-8) - Building**

#### **Katakana (46 characters)**
```
Introduced after hiragana mastery

Focus: Foreign words kids know
- アイス (aisu / ice cream) 🍦
- ケーキ (keeki / cake) 🍰
- テレビ (terebi / TV) 📺
- ゲーム (geemu / game) 🎮

Side-by-side comparison:
┌──────────┬──────────┐
│ Hiragana │ Katakana │
├──────────┼──────────┤
│ あ (a)   │ ア (a)   │
│ い (i)   │ イ (i)   │
│ う (u)   │ ウ (u)   │
└──────────┴──────────┘
```

#### **Simple Sentences (N5 Grammar)**
```
Pattern: [Subject] は [Object] です
(wa / is)

Examples shown in all formats:

これはねこです。
これ は ねこ です。
kore wa neko desu.
This is a cat.

わたしはがくせいです。
わたし は がくせい です。
watashi wa gakusei desu.
I am a student.
```

#### **Vocabulary (300 words)**
```
Daily life:
- School: がっこう、せんせい、ともだち
- Time: あさ、ひる、よる、きょう、あした
- Actions: たべる、のむ、みる、きく、いく
- Places: うち、こうえん、スーパー
```

### **Level 3: Late Elementary (Ages 9-10) - Advancing**

#### **Basic Kanji (80 characters)**
```
Introduced with stories and mnemonics

Example: 木 (ki / tree)
┌─────────────────────────────┐
│        木                   │  ← Kanji
│        き                   │  ← Hiragana reading
│        ki                   │  ← Romaji
│        Tree                 │  ← English
│                             │
│   Looks like a tree! 🌳     │  ← Mnemonic
│   Trunk (|) + Branches (—)  │
│                             │
│   [🔊] [✏️ Trace] [🎮 Game] │
└─────────────────────────────┘

JLPT N5 Kanji:
一、二、三、四、五、六、七、八、九、十
日、月、火、水、木、金、土、人、子、女
山、川、田、目、耳、口、手、足、犬、猫
```

#### **Grammar Patterns (N5-N4)**
```
Taught through interactive stories:

Pattern: 〜たい (want to)
たべたい (tabetai / want to eat)
のみたい (nomitai / want to drink)

In-game:
NPC: "なにがたべたいですか？"
      (nani ga tabetai desu ka?)
      (What do you want to eat?)

Child selects:
[🍎 りんご] [🍰 ケーキ] [🍙 おにぎり]

Response: "りんごがたべたいです！"
         (ringo ga tabetai desu!)
         (I want to eat an apple!)
```

### **Level 4: Parent Mode (N4-N3)**
**Adults playing with kids can practice advanced content**

```
While child learns basics, parent sees:

Child's screen:
┌─────────────────────────────┐
│   ねこ (neko) / Cat 🐱      │
│   [Simple task]             │
└─────────────────────────────┘

Parent's screen:
┌─────────────────────────────┐
│   猫を飼っています。         │

│   neko wo katteimasu.       │
│   I have a cat.             │
│   [Advanced grammar task]   │
└─────────────────────────────┘

Both contribute to same goal!
```

### **Level 5: Advanced (N2-N1)**
**Optional for motivated older kids or adults**

```
Unlocked after N4 completion

Content:
- Complex kanji (常用漢字)
- Keigo (polite language)
- Idioms (慣用句)
- Reading comprehension
- Writing practice
```

### **Multilingual Text Widget**
```dart
class MultilingualText extends StatelessWidget {
  final String kanji;
  final String hiragana;
  final String romaji;
  final String english;
  final String audioFile;
  final bool showKanji;
  final TextStyle style;
  
  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        if (showKanji && kanji.isNotEmpty)
          Text(kanji, style: style.copyWith(fontSize: 32)),
        Text(hiragana, style: style.copyWith(fontSize: 24, color: Colors.red)),
        Text(romaji, style: style.copyWith(fontSize: 18, color: Colors.blue)),
        Text(english, style: style.copyWith(fontSize: 16)),
        IconButton(
          icon: Icon(Icons.volume_up, size: 32),
          onPressed: () => AudioService.play(audioFile),
        ),
      ],
    );
  }
}

// Usage:
MultilingualText(
  kanji: '猫',
  hiragana: 'ねこ',
  romaji: 'neko',
  english: 'Cat',
  audioFile: 'neko.mp3',
  showKanji: playerAge >= 9,
)
```

### **Age-Appropriate Content Filter**
```dart
class ContentManager {
  static List<VocabularyItem> getVocabulary(int age) {
    if (age <= 6) {
      return vocabularyN5Basic; // Colors, numbers, animals
    } else if (age <= 8) {
      return vocabularyN5Full; // Daily life, simple verbs
    } else {
      return vocabularyN5N4; // Include grammar, kanji
    }
  }
  
  static bool shouldShowKanji(int age) {
    return age >= 9;
  }
```
