### **2. Fun Mnemonics (Memory Tricks) 🧠**
Each kanji has a **story or trick** to help you remember!

**Examples:**
- **一** = "ONE horizontal line = 1"
- **二** = "TWO horizontal lines = 2"
- **三** = "THREE horizontal lines = 3"
- **四** = "FOUR corners in a box"
- **五** = "FIVE fingers working together"
- **八** = "EIGHT splits apart ∧"
- **十** = "TEN is a cross + symbol"

---

### **3. Hiragana Associations 🔤**
See how kanji connects to hiragana you already know!

**Examples:**
- **日** → ひ / にち
- **月** → つき / げつ
- **火** → ひ / か
- **水** → みず / すい
- **木** → き / もく
- **人** → ひと / じん

Now you can **link kanji to sounds** you're familiar with!

---

### **4. Real Japanese Examples 📝**
See kanji used in actual Japanese words!

**Examples:**
- **一人** (ひとり) = one person
- **今日** (きょう) = today
- **月曜日** (げつようび) = Monday
- **火曜日** (かようび) = Tuesday
- **水曜日** (すいようび) = Wednesday
- **木曜日** (もくようび) = Thursday
- **金曜日** (きんようび) = Friday
- **土曜日** (どようび) = Saturday
- **日本人** (にほんじん) = Japanese person

---

### **5. Stroke Order Learning ✏️**
Learn **how to write** kanji correctly!

**Examples:**
- **一** = → (1 stroke)
- **二** = → → (2 strokes)
- **三** = → → → (3 strokes)
- **日** = | → | → (4 strokes)
- **十** = 一 | (2 strokes - horizontal then vertical)

Each kanji shows the **stroke sequence** so you can practice writing!
