## 🔄 **HOW RANDOMIZATION WORKS:**

### **Algorithm:**
```javascript
1. User selects category (e.g., "Animals")
2. System shuffles all 10 animal words
3. Selects first 8 words randomly
4. Creates 16 cards (8 pairs):
   - Card A: Japanese word
   - Card B: Emoji + English
5. Shuffles all 16 cards
6. Displays on board
```

### **Random Mode:**
```javascript
1. User clicks "🎲 ランダム"
2. System picks random category
3. Follows same algorithm
4. Each game different category possible
```

### **Benefits:**
- ✅ Never same game twice
- ✅ Reinforcement through repetition
- ✅ Natural spaced repetition
- ✅ Maintains interest
- ✅ Encourages replay
