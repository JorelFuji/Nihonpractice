✅ Memory Match Rules

  • • Match Japanese word with picture
  • • Match hiragana with romaji
  • • Match word with English meaning
  • • Learn 5-10 words per game
  • • Repeat until mastered

  • • 🍎 Food (たべもの)
  • • 🐱 Animals (どうぶつ)
  • • 🏠 Home (いえ)
  • • 🎨 Colors (いろ)

✅ Progress Tracking

  • • Track words learned
  • • Record high scores
  • • Monitor accuracy rate
  • • Set daily goals
  • • Celebrate achievements

  • • Stage 2: Katakana (46)
